# A Jornada de Kuki por Brusque

## Sobre o Projeto

A Jornada de Kuki por Brusque é um jogo educativo infantil desenvolvido para a disciplina de Curricularização da Extensão 2026.1.

O objetivo do jogo é ensinar crianças sobre a história, cultura e pontos turísticos da cidade de Brusque de forma lúdica, visual e interativa.

O personagem principal é Kuki, um marreco que guia as crianças em uma jornada educativa pela cidade.

---

## Funcionalidades

- Tela inicial interativa
- Navegação entre perguntas
- Interface visual infantil
- Perguntas ilustradas
- Estrutura preparada para múltiplas perguntas
